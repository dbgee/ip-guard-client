## 代码贡献
1. Fork [本项目](https://github.com/dbgee/ip-guard-client) 到自己的github 账号下
2. Clone 自己的项目代码到本地
3. 在本地修改代码（修复bug、新增功能）
4. push 到自己的项目代码下
5. PR (pull requests) 到本项目


## Project setup

npm install
